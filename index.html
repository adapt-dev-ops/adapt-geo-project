MD 세팅 필요 건
MD
1
ALT 태그 기입
상세페이지 분할 후 이미지별 알트 태그 작성. 150자 내외, 성분명+기능+피부타입 포함.
2
메타 데이터 설정
Meta title / Meta description 작성 후 카페24 SEO 설정에 직접 입력.
3
상세 FAQ 취합
CS 빈도 높은 내용 가공. USP당 1~2개 구성, 완결된 문장으로 답변.
4
성과 분석
Google Rich Results Test + Beusable GEO로 스키마 점수 확인. 클로드 활용 자가진단 프롬프트 병행.
개발파트 세팅 필요 건
개발
1
H1 / H2 태그
상품 페이지의 제목 계층 구조를 정의. 검색엔진·AI 모두가 콘텐츠 구조를 파악하는 핵심 요소.
2
FAQ 탭 개설
CS 빈도 높은 질문 5~7개를 텍스트로 작성 + FAQPage 스키마 구조화. AI 직접 인용 가능성 극대화.
3
통합 상세 스키마 설정 점검
Product / FAQ / Review / Breadcrumb 스키마 점검. @graph 구조로 통합 관리.
4
연관 상품 노출 구간 생성
상세 페이지 하단 내부 링크 구조. ItemList 스키마로 연관 상품 구조화.
5
메인페이지 스키마 작업
메인·서브 페이지 Organization + WebSite 스키마 적용.
6
매거진 탭 신설
콘텐츠 내용은 PR 파트 확인 필요. 백링크 고려.
