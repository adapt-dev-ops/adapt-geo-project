GEO 2단계 검증 인사이트

자사 미노출 상품 대상 · ChatGPT · Gemini · Perplexity 직접 검색 · 2026년 7월

핵심 결론

스키마·백링크는 AI 추천의 직접 결정 요인이 아님. 외부 언급량(블로그·유튜브·커뮤니티)과 브랜드 누적 언급 이력이 AI 추천을 결정한다. 자사가 AI에 등장하려면 제품 개선이 아닌 외부 콘텐츠 볼륨 확보가 최우선 과제.

1. 핵심 발견

① 스키마 ≠ AI 추천 결정요인
타사(라운드랩·다슈) 스키마는 FAQPage·Organization·HowTo 전무, description도 해시태그 4개 수준. 그럼에도 모든 AI에서 1위 추천.
→ 스키마 최적화는 AI 노출의 직접 해법이 아님
② 외부 언급량 격차가 핵심
풀리 vs 라운드랩 — 화해 리뷰 454개 vs 29,231개, 유튜브 구독자 0 vs 127K. AI는 웹상의 언급 빈도 기반으로 추천.
→ 자발적 UGC 볼륨 확보가 최우선 과제
③ 제품 성과 ≠ AI 추천
오브제는 올리브영 1위·무신사 821개지만 AI에서 미노출. 다슈는 11년 연속 소비자만족지수 1위로 쌓인 누적 언급량이 AI에 이미 학습됨.
→ 브랜드 누적 언급량이 AI 추천을 결정
④ PR 사이클 부재
타사: 수상 → 수상 기사(고권위) → 블로그 반응 → 커뮤니티 → AI 학습 사이클이 반복. 자사는 이 사이클 자체가 없음.
→ 어워즈 수상 → PR 기사 확보가 출발점

2. 외부 언급량 비교

풀리 선크림 vs 라운드랩 자작나무 수분 선크림

결론

풀리 제품 만족도(무신사 4.9점)는 높지만 자발적 UGC 볼륨이 거의 없다. 화해 리뷰 29,231개·올리브영 48,061건인 라운드랩과 모든 채널에서 격차가 압도적. AI는 스키마가 아닌 웹상의 언급 빈도를 기반으로 추천.

3. PR · 기사 비교

결론

라운드랩·다슈가 AI에서 1위인 구조: 수상 → 수상 기사(고권위) → 블로그 반응(중권위) → 커뮤니티(중권위) → AI 학습 반복. 자사는 이 사이클 자체가 없다. 블로그·유튜브보다 뉴스 기사 권위가 높기 때문에 어워즈 수상 전략이 최우선.

4. 백링크 분석

결론

오브제(DR 11) · 다슈(DR 11) — DR 동일, dofollow 참조 도메인도 유사. 그럼에도 AI 결과 역전: 다슈 1위, 오브제 미노출. 백링크는 AI 추천의 직접 결정 요인이 아님. 고권위 링크 → 크롤러 수집 우선도 → AI 학습 포함 확률 증가라는 간접 경로만 존재.

직접 연관 없음

AI는 링크 그래프 구조를 직접 학습하지 않음. 백링크 수치는 AI 추천 알고리즘의 입력 변수가 아님. PBN·스팸 링크 매입으로 DR을 높여도 GEO에 효과 없음.

간접 연관 있음 (부산물)

수상·PR → 고권위 미디어 기사 → 자연 링크 누적 → 크롤러 수집 우선도 증가 → AI 학습 포함 확률 상승. 백링크는 외부 언급량 확대의 결과물.
